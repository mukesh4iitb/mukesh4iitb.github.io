VASP:

I have developed VaspWorks, a workflow framework based on Python and Bash. It automates input generation for various VASP simulations, similar to VASPkit.

  • Structure relaxation
  • Density of States (DOS)
  • Band structure calculations
  • Selective dynamics
  • Nudged Elastic Band (NEB)

🔗 Repository: VaspWorks

Additionally, the repository includes several pre- and post-processing tools:

  • DOS and band structure plotting
  • Molecular dynamics (MD) analysis plots
  • Archiving VASP output files

🔗 Repository: pre_and_post_processing_dft/vasp

VestaPy:

VestaPy is a Python-based automation tool for VESTA that eliminates manual work. It allows automation of:

  • Atom color customization
  • Bond color modification
  • Bond-length adjustments
  • Atom size control
  • General visualization automation

Most operations that are manually performed in VESTA are supported. New features are continuously added.

🔗 Repository: VestaPy

RxnPy:

RxnPy is a toolkit for studying reaction energetics and adsorption on surfaces.

  • Polysulfides Adsorbed structure Generator: Generates structures of polysulfides adsorbed on surfaces. It can be adapted for other molecules on various surfaces.
  • Energy Analysis: Calculates adsorption/absorption energies and Gibbs free energies of given reactions (Reaction.in and Reaction1.in files). It can be extended to any kind of reactions.

🔗 Repository: RxnPy
